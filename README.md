# hexit Landing Page

Multilingual landing page for [hexit](https://github.com/keinEntwickler/hexit), an iOS color guessing game.

Live at: [keinentwickler.github.io/hexit-landing-page](https://keinentwickler.github.io/hexit-landing-page/)

## Languages

- English (default, `/`)
- Deutsch (`/de/`)
- Italiano (`/it/`)
- Francais (`/fr/`)
- Espanol (`/es/`)
- Chinese (`/zh-Hans/`)

## Structure

- **Landing pages** — one per language, content from `_data/translations/`
- **Legal pages** — single multilingual documents (Privacy Policy, Legal Notice)
- **Open Source** — credits for Swift frameworks used in hexit

## Local Development

```bash
bundle install
bundle exec jekyll serve
```
